## 📦 包含的文件

- `exported-data.json` - 完整的题目数据（JSON 格式）

### 2. 图片文件
- `downloaded-images/` - 所有题目和答案的图片文件
  * 共 5 个图片文件
  * 总大小约 1.5 MB

### 3. 迁移脚本
- `export-database.ts` - 数据库导出脚本（已执行）
- `download-images.ts` - 图片下载脚本（已执行）

### 4. 迁移指南
- `MIGRATION_GUIDE.md` - 详细的迁移步骤说明

## 🚀 快速迁移步骤

### 第一步：下载迁移包

将整个 `migration/` 文件夹下载到本地。

### 第二步：在新账号创建项目

在新 Manus 账号的新对话中说：

```
从 GitHub 导入项目：[您的 GitHub 用户名]/math-ai-tutor
```

### 第三步：导入数据库

在新项目创建后，在对话中说：

```
帮我导入数据库。我有一个 SQL 文件。
```

然后上传 `import-problems.sql` 文件。

### 第四步：上传图片

**方法一：使用管理界面（推荐）**

1. 打开项目管理界面
2. 将 `downloaded-images/` 文件夹中的图片复制到新项目的 `client/public/uploads/` 目录

**方法二：在对话中上传**

在对话中说：

```
我需要将这些图片上传到 client/public/uploads/ 目录
```

然后逐个上传 `downloaded-images/` 中的图片。

### 第五步：验证

访问新项目的首页，确认：
- ✅ 题目列表显示正常
- ✅ 题目图片正常显示
- ✅ 点击题目可以查看详情
- ✅ 苏格拉底模式正常工作

## ⚠️ 重要提示

1. **图片路径**: 图片使用相对路径（`/uploads/...`），存储在 `client/public/uploads/` 目录
2. **数据库**: 使用 MySQL/TiDB，字符集为 UTF-8
3. **用户数据**: 本迁移包不包含用户学习记录（`userProgress` 表）
