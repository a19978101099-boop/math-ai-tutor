# Bug 分析：AI 提示不显示

## 问题描述
点击步骤后，右侧 AI 提示面板显示"选中步骤"提示，但点击"为什么会得到这一步？"或"下一步怎么思考？"按钮后，没有显示 AI 提示内容。

## 浏览器测试结果
1. 点击步骤后，右侧面板正确显示了选中步骤的内容
2. 显示了"为什么会得到这一步？"和"下一步怎么思考？"两个按钮
3. 但是没有看到 AI 提示的文本内容

## 可能的原因
1. `currentHint` 状态没有正确更新
2. hint API 调用失败
3. UI 渲染逻辑有问题（可能是条件渲染导致内容不显示）
4. 权限问题：hint API 可能需要登录但改成了 publicProcedure

## 下一步检查
1. 检查 ProblemDetail.tsx 中 hint 显示的 UI 逻辑
2. 检查 hint API 的权限设置
3. 检查浏览器控制台是否有错误
